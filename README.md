# EEG Commander App
A generic and declarative command feature, for interfacing with external devices, to avoid having to code every possible API integration scenario into an app.

EEG signal patterns are mapped to common action types, like OS/Shell, Url, JavaScript, WebSocket, UDP, MQTT and SDK functions, supporting most CLI- & common Web API based interfaces.

## Commander
A web app used to run commands based in signal patterns in the EEG data stream.
[See more](/commander/)

## Shell Server
A node.js app used to execute OS/Shell commands.
[See more](/shell/)

## WebSocket Echo Server
A node.js app used to echo socket commands.
[See more](/socket/)

## UDP Echo Server
A node.js app used to echo udp commands.
[See more](/udp/)
