# Supported Browsers
For the EEG Commander app to connect to an EEG device the browser has to support the Bluetooth Web API or in some other way supply EEG data.

If you're only using the EEG simulator, every newer browser is supported.

## Browsers currently supporting the Bluetooth Web API (for using EEG devices)
- Chrome
- Edge
- Opera

## Without an EEG device
If you don't have a supported EEG device, you can just use the EEG simulator to play with your patterns.
