# Supported Browsers
For the EEG Commander app to connect to an EEG device the browser has to support the Bluetooth Web API.

## Browsers supporting Bluetooth Web API
- Chrome
- Edge
- Opera

## Browsers not supporting Bluetooth Web API
- Firefox

## Without an EEG device
If you don't have a supported EEG device, you can just use the EEG simulator to play with your patterns.
