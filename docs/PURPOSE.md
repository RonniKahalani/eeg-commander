# The Purpose
The purpose of the EEG Command idea is to:
- Ignite EEG creativity and excitement
- Create added value for EEG device consumers
- Nurture a fun and playful [open-source community](/docs/COMMUNITY.md)

## Ignite EEG creativity and excitement
- Facilitate a culture for creating innovative EEG experiments, like:
- Control a drone- or IoT device with the mind
- Make a change in the current user interface (UI)
- Talk to the user and play sounds
- Show notifications or sebd data to external sources

## Create added value for EEG device consumers
- Give EEG users higher return on their investment in devices
- Make it fun creating their own EEG actions.

## Nurture a fun and playful open-source community
- Make it fun and playful for tech nerds, developers and consumers to ideate on new ways of creating real-world actions from brain signal patterns.
- Plan hackathons, TechTalks, competitions and open ideation sessions for the community.
